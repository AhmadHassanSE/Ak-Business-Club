1. Project Overview
- Business Name: AK Traders
- Business Type: Food items (ketchup, mayonnaise, kabab, etc.)
- Website Type: Full-stack web application
- Interfaces:
- Admin Panel: Manage products (add, update, delete, set prices).
- User Interface: Browse menu, place orders, orders sent to business email (hranaahmad82@gmai.com).

2. Functional Requirements
Admin Panel
- Add new food items (name, description, price, image).
- Update existing items (price, description, availability).
- Delete items from menu.
- View list of all items.
- Authentication (login/logout with secure credentials).
User Interface
- Browse menu with categories (e.g., sauces, kababs, etc.).
- Search and filter items.
- Add items to cart.
- Place orders (checkout form with name, phone, address).
- Orders sent to admin email.
- Confirmation message after order placement.
Additional Functionalities
- Responsive design (mobile + desktop).
- Image upload for products.
- Basic analytics (track number of orders, popular items).
- Contact form for customer queries.
- About Us page with business details.

3. Non-Functional Requirements
- Performance: Pages load within 2 seconds.
- Security: Passwords hashed, admin panel protected.
- Scalability: Support at least 1000 users concurrently.
- Usability: Simple navigation, clear UI.
- Compatibility: Works on Chrome, Firefox, Edge, Safari, mobile browsers.
