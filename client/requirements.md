## Packages
framer-motion | Page transitions and complex animations
zustand | Client-side state management for the shopping cart
react-hook-form | Efficient form handling
@hookform/resolvers | Zod integration for react-hook-form

## Notes
- Tailwind config should include 'font-display' (Outfit) and 'font-body' (Inter/DM Sans)
- Cart state is persisted in localStorage via Zustand
- Admin routes are protected via a wrapper component checking useAuth
- Images utilize Unsplash placeholders where dynamic data is missing
